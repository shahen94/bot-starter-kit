# Bot Framework + LUIS = :heart:

## Getting Started
##
#### Create .env file and add the following variables

`PORT` -> server port
`NODE_ENV` -> Enviorment
`API_VERSION` -> set API version,(i.g. `v1`, `v2`,)
`BOT_APP_ID` -> Bot App ID
`BOT_APP_PASSWORD` -> Bot App Password
`LUIS_MODEL` -> LUIS app's endpoint

```sh
$ npm install
```

```sh
$ npm run watch
```
